Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при
передаче их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}

```

Ответ:

Программа выведет [3 2 3].
Срез s в мейне передается в функцию modifySlice().
Внутри функции modifySlice() изменяется содержимое среза. Но после выполнения функции modifySlice() измененный срез не
возвращается, поэтому он остается неизменным за пределами функции.
Поэтому содержимое среза s остается неизменным.

Срез состоит из трех компонентов:

1. Указатель на массив - указывает на первый элемент в массиве слайса.
2. Длина слайса - это количество элементов в слайсе.
3. Емкость слайса - это количество элементов в базовом массиве, начиная с первого элемента слайса.

При передаче слайса в функцию в качестве аргумента создается копия указателя на массив и копии длины и емкости слайса.
Таким образом, функция получает доступ к тому же массиву данных, что и вызывающая функция, и может изменять содержимое
этого массива. Однако, если функция изменяет длину или емкость слайса, эти изменения не отражаются за пределами функции.
